# SCBKR Roadmap

## 1.0 Release Candidate

0.15.0-rc.1 對應 P15-Q Release Candidate + PATCH-2，重點是責任鏈、簽名、驗收、二次確認、Data Center 與四庫 target 收束。

## P15-R Documentation and product readiness

整理 README、README_EN、Release Notes、安裝文件、模型設定、手機連回本機、隱私安全與圖片索引，作為 1.0 上線基礎。

## P15-R-I18N Bilingual UI

繁體中文 / English UI 語言包基礎。此項為下一步，不屬於 P15-R-DOC 實作範圍。

## P15-R-SG01 Semantic Legality Gate

Semantic Legality Gate 屬於後續規劃，不屬於目前 1.0 已完成功能。

## P15-R-SEARCH Web Search Candidate Flow

Web Search Candidate Flow 屬於後續規劃，不是目前 production feature。

## P16 Rule Design Engine

Rule Design Engine 將作為未來規則設計能力規劃，不屬於 1.0 core。

## P16 Tool Permission Gate

Tool Permission Gate 將管理工具使用權限與授權邊界，仍屬未來工作。

## 2.0 Rule-driven AI Engine

2.0 目標包含 RulePack、工具權限、候選搜尋、語音、影像、團隊治理與企業級責任鏈治理。以上皆為 Roadmap / Future Work。
