# Store Release Checklist

- Local-first runtime starts.
- Chat, Workbench, Rule Center, and Data Center load.
- Rule generation creates editable S/C/B/K/R.
- Model cannot sign, store, or activate.
- User signature changes status to owner-signed.
- Review and storage confirmation compile an Active rule.
- Later answers use `current_rule_package.chat_context_used = false`.
- Token / Cost Audit reports actual compression.
- Mobile layout does not overlap.
- Build, pytest, and UI tests pass.
