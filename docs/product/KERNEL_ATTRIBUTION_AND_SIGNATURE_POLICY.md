# Kernel Attribution And Signature Policy

Kernel structure source: 許文耀 / 沈耀 SCBKR Kernel.

The Kernel provides structure and validation. It does not make the local user's
rule automatically valid. A rule becomes formally usable only after:

- S/C/B/K/R draft exists.
- User edits or accepts the draft.
- User signature is recorded.
- Review passes.
- Storage is confirmed.
- The compiled rule is Active.

Model outputs are always draft-only before user signature.
